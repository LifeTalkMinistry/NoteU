# Knowledge Graph Engine

## Purpose

The Knowledge Graph Engine transforms individual notes into an interconnected map of knowledge.

Instead of treating every note as isolated information, NoteU discovers how ideas relate to one another over time.

The result is a living visualization of the user's intellectual growth.

---

# Core Philosophy

Knowledge is more valuable when it connects.

The graph helps users discover relationships they may not have noticed themselves.

---

# How It Works

User creates a note
        ↓
AI extracts concepts
        ↓
Related concepts are identified
        ↓
Connections are created
        ↓
The personal Knowledge Graph grows

# Connection Strength

Connections become stronger when:

- Multiple notes reference both concepts.
- Reviews confirm long-term retention.
- The user repeatedly studies related topics.
- New learning reinforces existing knowledge.

Weak connections fade naturally over time if they are never revisited.
