# NoteU Identity Score System

## Purpose

The Identity Score measures how strongly a user's notes, reviews, and retained knowledge reveal their learning identity.

It does not reward random note quantity. It rewards meaningful learning, consistency, review completion, and knowledge connection.

## Core Formula

Identity Score = Learning Depth + Consistency + Review Retention + Knowledge Diversity + Knowledge Combination

## Score Factors

### 1. Learning Depth
Measures how meaningful and detailed the user's notes are.

Signals:
- Clear explanation
- Personal understanding
- Examples
- Original reflection
- Not copied randomly

### 2. Consistency
Measures how often the user learns and records notes.

Signals:
- Daily or weekly note activity
- Learning streaks
- Long-term progress

### 3. Review Retention
Measures whether the user actually remembers what they wrote.

Signals:
- Daily 30-minute review completion
- Quiz performance
- Recall accuracy
- Missed review penalty

### 4. Knowledge Diversity
Measures how many fields the user is learning from.

Example fields:
- Science
- Finance
- Psychology
- Business
- Faith
- Technology
- Creativity
- Leadership

### 5. Knowledge Combination
Measures how different fields connect to form a stronger identity.

Examples:
- Finance + Psychology = Behavioral Finance Learner
- Faith + Leadership = Faith-Based Leader
- Creativity + Strategy = Creative Strategist
- Science + Technology = Innovation Thinker

## Score Behavior

Skipping reviews lowers the score because NoteU cannot verify retention.

Writing many notes without review does not create a strong identity.

A strong Identity Score requires:
- Honest notes
- Consistent review
- Retained knowledge
- Cross-field learning

## Product Principle

NoteU should never make users feel judged.

The score should feel like a mirror, not a punishment.

It should help users see who they are becoming.
