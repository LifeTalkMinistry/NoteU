# NoteU Product Requirements Document

## Product Name

NoteU

## Tagline

The notes that reveal you.

## Product Summary

NoteU is a learning identity app that analyzes what users write, review, retain, and connect over time.

Unlike traditional note apps that only store information, NoteU helps users discover who they are becoming through consistent learning.

## Problem

People collect notes but often do not know:

- What they truly understand
- What they have retained
- What fields they are growing in
- How their knowledge connects
- What kind of learner or thinker they are becoming

## Solution

NoteU turns notes into a measurable Learning Identity Profile using:

- AI categorization
- Daily review
- Retention tracking
- Knowledge graph mapping
- Identity scoring
- Personalized learning insights

## MVP Scope

The first version should include:

- User onboarding
- Note creation
- AI note categorization
- Daily review system
- Identity Score
- Basic Learning Identity Profile
- Simple dashboard
- Basic Knowledge Graph

## Core User Flow

1. User opens NoteU.
2. User creates their learning profile.
3. User writes their first note.
4. AI categorizes the note.
5. NoteU extracts concepts.
6. User sees early learning identity.
7. User completes daily review.
8. Identity Score updates.
9. Knowledge Graph grows.
10. Dashboard shows progress.

## Core Screens

- Splash Screen
- Welcome Screen
- Onboarding
- Home Dashboard
- Note Editor
- AI Insight Screen
- Daily Review
- Knowledge Graph
- Identity Profile
- Search
- Settings

## Success Metrics

- Notes created per week
- Daily review completion rate
- User retention after 7 days
- User retention after 30 days
- Identity Score engagement
- Knowledge Graph engagement
- Returning users after first review

## Product Principle

NoteU should never feel like a school grading system.

It should feel like a mirror.

The goal is not to judge the user.

The goal is to reveal their growth.
