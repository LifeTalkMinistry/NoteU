# Learning Identity Engine

## Purpose

The Learning Identity Engine is the core system of NoteU.

It turns notes, reviews, retention, and knowledge combinations into a personal Learning Identity Profile.

The goal is not to label the user permanently.

The goal is to show what kind of thinker they are becoming.

## Main Inputs

The engine analyzes:

- Notes written by the user
- Categories of knowledge
- Review completion
- Retention performance
- Learning consistency
- Connections between topics
- Long-term learning patterns

## Identity Profile

The Learning Identity Profile may include:

- Primary Learning Field
- Secondary Learning Fields
- Strongest Knowledge Combinations
- Learning Depth Level
- Retention Strength
- Consistency Level
- Emerging Identity Title

## Example Identity Titles

- Scientific Thinker
- Behavioral Finance Learner
- Faith-Based Leader
- Creative Strategist
- Technology Builder
- Business Systems Thinker
- Reflective Communicator
- Practical Problem Solver

## Identity Levels

### Emerging
The user is beginning to show learning patterns.

### Developing
The user is consistently learning and reviewing.

### Established
The user has strong evidence of retained knowledge.

### Advanced
The user shows deep, consistent, cross-domain learning.

## Identity Rules

NoteU should not assign strong identities too early.

A user must show repeated evidence before receiving a higher identity level.

The system should prioritize honesty, retention, and consistency over quantity.

## Product Philosophy

Your notes are the evidence.

Your reviews are the verification.

Your identity is the pattern.
