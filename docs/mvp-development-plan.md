# NoteU MVP Development Plan

## Purpose

This document defines exactly what will be built for Version 1.

The MVP should prove one thing:

Can NoteU successfully help users build and recognize their Learning Identity?

Everything that does not help answer this question is postponed.

---

# MVP Objective

A user should be able to:

1. Write what they learned.
2. Review what they learned.
3. See how they are growing.
4. Stay motivated to continue learning.

---

# Version 1 Features

## User Profile

- Name
- Learning Goals
- Preferred Learning Fields

---

## Notes

- Create
- Edit
- Delete
- Search

Supported:

- Text
- Images

Future:

- Voice
- PDF
- Video

---

## AI Categorization

Automatically detect:

- Learning Domains
- Concepts
- Related Topics

---

## Daily Review

Generate questions from the user's own notes.

Support:

- Recall
- Explain
- Connect

---

## Identity Score

Calculate using:

- Learning Consistency
- Review Completion
- Retention
- Knowledge Diversity

---

## Learning Identity

Display:

- Identity Title
- Identity Level
- Growth Summary

---

## Dashboard

Cards:

- Today's Review
- Identity Score
- Learning Streak
- Recent Notes
- Learning Domains

---

## Knowledge Graph

Version 1

Simple interactive graph.

No advanced analytics yet.

---

## Local Backup

- Export
- Import
- Restore

No cloud sync.

---

# Not Included

Version 1 will NOT include:

- Community
- Shared Notes
- Cloud Sync
- Collaboration
- AI Chat Assistant
- Courses
- Marketplace
- Leaderboards
- Web Version

These features are intentionally postponed.

---

# MVP Success Criteria

Within the first month, users should say:

"I actually reviewed my notes."

"I remember more."

"I understand my strongest learning areas."

"I can see myself improving."

If users say these consistently, the MVP is successful.
