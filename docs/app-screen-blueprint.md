# NoteU App Screen Blueprint

## Design Philosophy

Every screen should answer one question:

"How does this help users become better learners?"

The interface should feel clean, intelligent, and encouraging—not overwhelming.

---

# 1. Splash Screen

Purpose

Introduce the brand.

Contents

- NoteU logo
- "The notes that reveal you."
- Soft animated network of connected knowledge nodes

---

# 2. Welcome Screen

Headline

Welcome to NoteU

Subtitle

Build your learning identity.

Buttons

- Get Started
- Sign In

---

# 3. Onboarding

Collect only essential information.

Questions

- What do you want to learn?
- Why are you learning?
- How much time can you study daily?

No more than 5 screens.

---

# 4. Home Dashboard

This is the heart of the app.

Cards

- Identity Score
- Current Learning Identity
- Daily Review Progress
- Learning Streak
- Knowledge Domains
- Recent Notes

Primary Button

+ New Note

---

# 5. Note Editor

Purpose

Capture understanding, not copied information.

Features

- Rich text
- Images
- Voice notes
- AI categorization
- Suggested related concepts

---

# 6. AI Insight Screen

After saving a note.

Displays

- Concepts
- Confidence
- Connected topics
- Suggested reviews

---

# 7. Daily Review

Today's review session.

Question types

- Recall

- Multiple Choice
- Reflection

Progress bar

30-minute target.

# Design Principles

Minimal.

Fast.

Beautiful.

Focused.

Learning first.

Everything on the screen should reinforce one idea:

You are becoming someone through what you consistently learn.
